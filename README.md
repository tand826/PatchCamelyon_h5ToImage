# PatchCamelyon_h5ToImage
Convert PatchCamelyon dataset to png images.

# Usage
```
python patch2img.py
```

# Dataset
- https://github.com/basveeling/pcam
- download script
	- `./download.sh`

# Requirements
```
pip install -r requirements.txt
```
